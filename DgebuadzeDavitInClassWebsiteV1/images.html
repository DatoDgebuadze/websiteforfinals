Before Creating a website you need to have some kind of workflow process, As a Developer before I start coding At first I always: Define the objectives and target audience, Plan the site structure

Behind every website there is some kind of CMS, What is CMS you may ask? CMS is a content managment system Which helps website administrators manage their website's efficiently and easily.

Price of my service is depended on what you want, The more complex the demand is higher is the price, It's based on how many libraries needs to be used, is CMS needed or not for the website.

I want my clients to know how I am doing what I am doing, so each of my client knows how their websites are built when I am saying how i mean what components are used for it to be built, For visual side whatever user sees in the browser I use Reactjs

As of Server side rendering, I use Nodejs Which is server side library, Mainly every hidden operations which cannot be seen by user or browser is done in nodejs. Which is the key part, brain of the website.
